# web ranking list

### 功能介绍

注册或登录以查看分数排行榜。学号（即账号）必须是12位，密码随意无限制。

初始用户：202300130062 密码：123

总分数降序排列，显示每道题得分并且通过为绿色没通过为红色

可点击刷新按钮刷新页面和数据

可点击添加数据按钮添加新数据，提交后会自动刷新

可点击修改按钮根据学号修改对应学生分数，提交后会自动刷新

用户信息和分数信息都是存储在mysql数据库里。
